🟡 HOW TO APPROACH IT

CORE INSIGHT:
→ Kafka clients support INTERCEPTORS — hooks that run before/after
  every produce and consume operation
→ Interceptors are configured via client properties — no code changes
→ They can modify messages, add headers, record metrics, inject tracing
→ Think of them as MIDDLEWARE for Kafka — like HTTP middleware for web servers

INTERCEPTOR ARCHITECTURE:

  WITHOUT INTERCEPTORS:
  ┌──────────┐              ┌───────┐              ┌──────────┐
  │ App Code │──produce()──▶│ Kafka │──poll()──────▶│ App Code │
  │          │              │       │              │          │
  └──────────┘              └───────┘              └──────────┘
  No tracing, no metrics, no correlation

  WITH INTERCEPTORS:
  ┌──────────┐  ┌────────────────┐              ┌────────────────┐  ┌──────────┐
  │ App Code │─▶│ PRODUCER       │──produce()──▶│ CONSUMER       │─▶│ App Code │
  │          │  │ INTERCEPTOR    │              │ INTERCEPTOR    │  │          │
  │          │  │                │              │                │  │          │
  │          │  │ • Add trace_id │   ┌───────┐  │ • Read trace_id│  │          │
  │          │  │ • Add timestamp│──▶│ Kafka │──▶│ • Calc latency │  │          │
  │          │  │ • Count msg    │   │       │  │ • Count msg    │  │          │
  │          │  │ • Log produce  │   └───────┘  │ • Log consume  │  │          │
  │          │  └────────────────┘              └────────────────┘  │          │
  └──────────┘                                                      └──────────┘
  App code UNCHANGED — interceptors handle all observability

TWO TYPES OF INTERCEPTORS:

  1. PRODUCER INTERCEPTOR
     → on_send(): Called BEFORE message is serialized and sent
       • Can modify the message (add headers, transform value)
       • Can reject the message (return None)
       • Perfect for: adding trace_id, timestamp, service_name
     
     → on_acknowledgement(): Called AFTER broker acknowledges (or fails)
       • Cannot modify message (already sent)
       • Perfect for: recording latency, counting success/failure

  2. CONSUMER INTERCEPTOR
     → on_consume(): Called AFTER message is received but BEFORE app processes
       • Can modify the message (add metadata, transform)
       • Perfect for: reading trace_id, calculating end-to-end latency
       • Perfect for: logging message receipt, counting consumption

     → on_commit(): Called AFTER offset is committed
       • Perfect for: tracking commit success/failure

    → Logs: "Received message trace=abc-123 from payment-service, latency=12ms"
    → When producing result:
      → PRESERVES trace_id = "abc-123" (same transaction)
      → Updates source_service = "fraud-service"
      → Updates produce_ts = current_time
      → Increments hop_count = "1"

  Order Service (Consumer Interceptor):
    → Reads trace_id = "abc-123"
    → Reads hop_count = "1" (passed through 1 intermediate service)
    → Calculates total latency from original produce_ts

✅ WHAT SHOULD DO

1. USE KAFKA HEADERS FOR ALL METADATA
   → Headers are the standard transport for trace context
   → Don't embed tracing in message value — pollutes business data
   → Headers survive serialization/deserialization unchanged
   → Compatible with OpenTelemetry and other tracing systems

2. GENERATE TRACE ID AT THE ENTRY POINT ONLY
   → First producer in the chain generates trace_id
   → All downstream services PROPAGATE the same trace_id
   → If consuming service also produces → copy trace_id to outgoing message
   → One trace_id per business transaction

3. MAKE INTERCEPTORS FAULT-TOLERANT
   → Interceptor failure must NEVER block message delivery
   → Wrap all interceptor logic in try/except
   → If tracing fails → log warning, send message without trace
   → Observability should OBSERVE, not OBSTRUCT

4. USE MONOTONIC TIMESTAMPS FOR LATENCY
   → time.time() can have clock skew between machines
   → For cross-service latency: use Kafka message timestamp
   → For same-machine latency: use time.monotonic()
   → Accept that cross-service latency is approximate

5. STANDARDIZE HEADER NAMES ACROSS ALL SERVICES
   → trace_id (not traceId, X-Trace-ID, correlation_id)
   → source_service
   → produce_timestamp_ms
   → hop_count
   → Document in a shared schema/contract

6. KEEP INTERCEPTOR OVERHEAD MINIMAL
   → Adding headers: ~0.1ms per message (negligible)
   → Metrics counting: in-memory, batched export (negligible)
   → Logging every message: can be expensive → sample at high volume
   → Target: < 1% overhead on produce/consume latency

❌ WHAT SHOULD NOT DO

1. NEVER LET INTERCEPTOR EXCEPTIONS CRASH THE APPLICATION
   → Interceptor throws NullPointerException → producer crashes
   → All messages stop flowing → production outage caused by tracing
   → Irony: observability tool causes the outage it was meant to prevent
   → ALWAYS catch all exceptions in interceptors

2. NEVER MODIFY THE MESSAGE VALUE IN INTERCEPTORS
   → Interceptors should add HEADERS, not change the payload
   → Changing the value breaks consumer deserialization
   → If you need to transform: use Kafka Streams (Project 20) or SMTs (Project 24)
   → Exception: adding envelope wrapper (but prefer headers)

3. NEVER LOG EVERY MESSAGE AT HIGH VOLUME
   → 500K msg/sec × log per message = disk full in minutes
   → Use sampling: log 1 in 1000 messages at high volume
   → Use counters: count all, log few
   → Use async logging: don't block produce/consume for I/O

4. NEVER GENERATE NEW TRACE ID FOR EVERY PRODUCE
   → Service consumes message with trace_id="abc"
   → Service produces result → MUST use trace_id="abc"
   → Generating new trace_id breaks the chain
   → Only the ENTRY POINT (first producer) generates trace_id

5. NEVER STORE LARGE DATA IN HEADERS
   → Headers add to message size
   → Large headers = larger messages = slower throughput
   → Keep headers small: IDs, timestamps, short strings
   → Don't put stack traces, full request bodies in headers

6. NEVER MAKE INTERCEPTOR DEPEND ON EXTERNAL SERVICE SYNCHRONOUSLY
   → Interceptor calls tracing API synchronously → 50ms added per message
   → At 100K msg/sec: 50ms × 100K = impossible
   → Use async export: buffer metrics, send in batches every 5-10 seconds
   → Or use local file/queue → separate exporter process

INTERCEPTORS vs ALTERNATIVES:

  Interceptors:
    → Transparent — no application code changes
    → Lightweight — runs in same process
    → Limited — can't do complex transformations
    → Best for: metadata injection, metrics, tracing

  SMTs (Project 24):
    → Runs in Kafka Connect
    → Can transform message content
    → Not available for application producers/consumers
    → Best for: data pipeline transformations

  Kafka Streams (Project 20):
    → Full stream processing
    → Can do complex transformations
    → Requires separate application
    → Best for: enrichment, aggregation, joins

⚙️ HOW IT ONLY WORKS (INTERNAL MECHANICS)

JAVA INTERCEPTOR LIFECYCLE (reference):

  Producer:
    1. Application calls producer.send(record)
    2. ProducerInterceptor.onSend(record) called
       → Returns modified (or same) ProducerRecord
       → Can add headers, modify key/value
    3. Serializer serializes key and value
    4. Partitioner assigns partition
    5. Record added to batch
    6. Batch sent to broker
    7. Broker acknowledges
    8. ProducerInterceptor.onAcknowledgement(metadata, exception) called
       → Receives success metadata or failure exception
       → Cannot modify — message already sent

  Consumer:
    1. Application calls consumer.poll()
    2. Broker returns messages
    3. Deserializer deserializes key and value
    4. ConsumerInterceptor.onConsume(records) called
       → Returns modified (or same) ConsumerRecords
       → Can add metadata, filter records
    5. Records returned to application
    6. Application processes records
    7. Application commits offsets
    8. ConsumerInterceptor.onCommit(offsets) called
       → Notified of committed offsets

HEADER ENCODING IN KAFKA:

  Headers are stored as List<Header> in the message:
  → Each header: (key: string, value: bytes)
  → Multiple headers with same key allowed
  → Headers stored UNCOMPRESSED (even if message compressed)
  → Headers included in message size calculation
  → Headers survive compaction (travel with the message)

  Size impact:
  → Header key: string (UTF-8 encoded)
  → Header value: bytes (application-defined encoding)
  → Typical trace headers: ~200 bytes total
  → For 500-byte message: 200/500 = 40% overhead
  → For 5KB message: 200/5000 = 4% overhead
  → Keep headers small for small messages

🕳️ COMMON PITFALLS AND FAILURE MODES

PITFALL 1: INTERCEPTOR EXCEPTION KILLS PRODUCER
  Symptom: Producer suddenly stops sending all messages
  Cause: Interceptor throws unhandled exception in on_send()
  Fix: Wrap ALL interceptor logic in try/except
  Prevention: Test interceptor with null values, empty headers, etc.

PITFALL 2: TRACE ID NOT PROPAGATED
  Symptom: Trace chain breaks at Service B
  Cause: Service B's producer interceptor generates NEW trace_id
         instead of reading from consumed message headers
  Fix: Interceptor checks for existing trace_id before generating new
  Pattern:
    on_produce():
      if context has trace_id from consumed message:
        use existing trace_id
      else:
        generate new trace_id (this is entry point)

PITFALL 3: CLOCK SKEW IN LATENCY MEASUREMENT
  Symptom: Negative latency values (-15ms)
  Cause: Producer machine clock ahead of consumer machine clock
  Fix:
    → Use Kafka's CreateTime (broker assigns if LogAppendTime)
    → Use NTP to synchronize clocks across machines
    → Accept ±10ms accuracy for cross-machine latency
    → Use relative timestamps for same-machine measurements

PITFALL 4: METRICS MEMORY LEAK
  Symptom: Interceptor process memory grows unbounded
  Cause: Storing per-message metrics in unbounded list/dict
  Fix:
    → Use bounded data structures (deque with maxlen)
    → Export and reset counters periodically
    → Use approximate data structures (HyperLogLog for unique counts)

PITFALL 5: HEADER ENCODING MISMATCH
  Symptom: Consumer reads garbled trace_id
  Cause: Producer encodes header as UTF-8, consumer reads as ASCII
  Fix: Standardize encoding: always UTF-8 for string headers
  Prevention: Use helper functions for header encode/decode

PITFALL 6: INTERCEPTOR ADDS TOO MUCH LATENCY
  Symptom: Produce latency increases by 50ms after adding interceptor
  Cause: Interceptor makes synchronous HTTP call to tracing service
  Fix:
    → Buffer metrics locally, export asynchronously
    → Use fire-and-forget for trace emission
    → Target: interceptor overhead < 0.1ms per message

📈 SCALABILITY CONSIDERATIONS

INTERCEPTOR OVERHEAD AT SCALE:

  Per-message overhead:
  → Header injection: ~0.01ms (dict update)
  → Counter increment: ~0.001ms (in-memory)
  → Timestamp read: ~0.001ms
  → Total: ~0.012ms per message

  At 500K messages/sec:
  → 0.012ms × 500,000 = 6,000ms of CPU per second
  → ~6 CPU-seconds per real second
  → On 8-core machine: ~75% of one core
  → Acceptable for most workloads

METRICS EXPORT STRATEGY:
  → Don't export per-message — export aggregated
  → Every 10 seconds: flush counters to metrics system
  → Counters: msg_count, bytes_count, error_count per topic
  → Histograms: latency percentiles (pre-computed)
  → Export via: Prometheus exposition, StatsD, file

TRACE SAMPLING:
  → At 500K msg/sec: tracing EVERY message is expensive
  → Sample rate: 1% (5,000 traces/sec) is usually sufficient
  → Critical traces: always trace (e.g., payment messages)
  → Sampling decision at entry point, propagated via header
  → Header: trace_sampled = "true" / "false"

class HeaderUtils:
    """
    Standardized header encoding/decoding.
    All interceptors use these to ensure consistency.
    
    Standard header names (contract across all services):
    → trace_id:            Unique ID for business transaction chain
    → correlation_id:      Business entity ID (e.g., order_id)
    → source_service:      Name of producing service
    → produce_timestamp_ms: Unix timestamp when produced (milliseconds)
    → hop_count:           Number of services message has passed through
    → trace_sampled:       Whether this trace should be recorded ("1"/"0")
    """

    TRACE_ID = 'trace_id'
    CORRELATION_ID = 'correlation_id'
    SOURCE_SERVICE = 'source_service'
    PRODUCE_TIMESTAMP = 'produce_timestamp_ms'
    HOP_COUNT = 'hop_count'
    TRACE_SAMPLED = 'trace_sampled'
    ORIGINAL_TOPIC = 'original_topic'

    @staticmethod
    def encode_header(value: str) -> bytes:
        """Encode string header value to bytes"""
        return value.encode('utf-8') if isinstance(value, str) else value

    @staticmethod
    def decode_header(value: bytes) -> str:
        """Decode bytes header value to string"""
        return value.decode('utf-8') if isinstance(value, bytes) else str(value)

    @staticmethod
    def get_header(headers: list, key: str) -> Optional[str]:
        """Get a header value by key from Kafka message headers"""
        if not headers:
            return None
        for h_key, h_value in headers:
            if h_key == key:
                return HeaderUtils.decode_header(h_value)
        return None

    @staticmethod
    def set_header(headers: list, key: str, value: str) -> list:
        """Set a header value (replace if exists, add if not)"""
        if headers is None:
            headers = []

        # Remove existing header with same key
        headers = [(k, v) for k, v in headers if k != key]
        # Add new header
        headers.append((key, HeaderUtils.encode_header(value)))
        return headers

    @staticmethod
    def headers_to_dict(headers: list) -> dict:
        """Convert Kafka headers list to dictionary"""
        if not headers:
            return {}
        result = {}
        for key, value in headers:
            result[key] = HeaderUtils.decode_header(value)
        return result

    @staticmethod
    def generate_trace_id() -> str:
        """Generate a new trace ID (UUID v4)"""
        return str(uuid.uuid4())

    @staticmethod
    def current_timestamp_ms() -> str:
        """Current time in milliseconds as string"""
        return str(int(time.time() * 1000))

class MetricsCollector:
    """
    In-memory metrics collection for interceptor telemetry.
    Thread-safe, bounded memory, periodic export.
    
    Collects:
    → Message counts per topic per direction (produce/consume)
    → Latency histograms per topic
    → Error counts
    → Trace chain information
    """

    def __init__(self, service_name: str, max_latency_samples: int = 10000):
        self.service_name = service_name
        self._lock = threading.Lock()

        # Counters
        self.produce_counts = defaultdict(int)
        self.consume_counts = defaultdict(int)
        self.error_counts = defaultdict(int)
        self.byte_counts = defaultdict(int)

        # Latency tracking (bounded)
        self.latencies = defaultdict(lambda: deque(maxlen=max_latency_samples))

        # Trace tracking
        self.active_traces = {}
        self.completed_traces = deque(maxlen=1000)

        # Timing
        self.start_time = time.time()
        self.last_export_time = time.time()

    def record_produce(self, topic: str, message_bytes: int = 0):
        with self._lock:
            self.produce_counts[topic] += 1
            self.byte_counts[f"produce:{topic}"] += message_bytes

    def record_consume(self, topic: str, message_bytes: int = 0):
        with self._lock:
            self.consume_counts[topic] += 1
            self.byte_counts[f"consume:{topic}"] += message_bytes

    def record_latency(self, topic: str, latency_ms: float):
        with self._lock:
            self.latencies[topic].append(latency_ms)

    def record_error(self, topic: str, error_type: str):
        with self._lock:
            self.error_counts[f"{topic}:{error_type}"] += 1

    def record_trace(self, trace_id: str, service: str, action: str,
                     topic: str, timestamp_ms: float):
        with self._lock:
            if trace_id not in self.active_traces:
                self.active_traces[trace_id] = []
            self.active_traces[trace_id].append({
                'service': service,
                'action': action,
                'topic': topic,
                'timestamp_ms': timestamp_ms
            })

    def get_percentile(self, topic: str, percentile: float) -> float:
        with self._lock:
            samples = list(self.latencies.get(topic, []))
        if not samples:
            return 0.0
        samples.sort()
        idx = min(int(len(samples) * percentile / 100), len(samples) - 1)
        return samples[idx]

    def get_metrics_snapshot(self) -> dict:
        """Get current metrics as a dictionary"""
        with self._lock:
            elapsed = time.time() - self.start_time

            snapshot = {
                'service': self.service_name,
                'uptime_seconds': round(elapsed, 1),
                'produce': dict(self.produce_counts),
                'consume': dict(self.consume_counts),
                'errors': dict(self.error_counts),
                'latency': {}
            }

            for topic, samples in self.latencies.items():
                if samples:
                    sorted_samples = sorted(samples)
                    count = len(sorted_samples)
                    snapshot['latency'][topic] = {
                        'count': count,
                        'p50': sorted_samples[int(count * 0.50)],
                        'p95': sorted_samples[int(count * 0.95)],
                        'p99': sorted_samples[min(int(count * 0.99), count - 1)],
                        'avg': sum(sorted_samples) / count,
                    }

            # Rates
            if elapsed > 0:
                snapshot['produce_rates'] = {
                    t: round(c / elapsed, 1) for t, c in self.produce_counts.items()
                }
                snapshot['consume_rates'] = {
                    t: round(c / elapsed, 1) for t, c in self.consume_counts.items()
                }

            return snapshot

    def print_metrics(self):
        """Print formatted metrics report"""
        snapshot = self.get_metrics_snapshot()

        print(f"\n  {'='*60}")
        print(f"  📊 INTERCEPTOR METRICS: {self.service_name}")
        print(f"  {'='*60}")
        print(f"  Uptime: {snapshot['uptime_seconds']}s")

        if snapshot['produce']:
            print(f"\n  📤 PRODUCE COUNTS:")
            for topic, count in snapshot['produce'].items():
                rate = snapshot.get('produce_rates', {}).get(topic, 0)
                print(f"     {topic:<30s} {count:>8,} ({rate:.1f}/sec)")

        if snapshot['consume']:
            print(f"\n  📥 CONSUME COUNTS:")
            for topic, count in snapshot['consume'].items():
                rate = snapshot.get('consume_rates', {}).get(topic, 0)
                print(f"     {topic:<30s} {count:>8,} ({rate:.1f}/sec)")

        if snapshot['latency']:
            print(f"\n  ⏱️  LATENCY (produce-to-consume):")
            for topic, stats in snapshot['latency'].items():
                print(f"     {topic}: p50={stats['p50']:.1f}ms "
                      f"p95={stats['p95']:.1f}ms p99={stats['p99']:.1f}ms "
                      f"avg={stats['avg']:.1f}ms (n={stats['count']})")

        if snapshot['errors']:
            print(f"\n  ❌ ERRORS:")
            for key, count in snapshot['errors'].items():
                print(f"     {key}: {count}")

        print(f"  {'='*60}")

import time

class InstrumentedProducer:
    """
    Drop-in replacement for confluent_kafka.Producer.
    Transparently adds:
    → trace_id header (new or propagated)
    → produce_timestamp_ms header
    → source_service header
    → hop_count header
    → Message counting metrics
    → Delivery latency tracking
    
    Usage:
      # BEFORE (no tracing):
      producer = Producer({'bootstrap.servers': 'localhost:9092'})
      
      # AFTER (with tracing — ONLY this line changes):
      producer = InstrumentedProducer(
          {'bootstrap.servers': 'localhost:9092'},
          service_name='payment-service'
      )
      
      # ALL existing produce() calls work unchanged
    """

    def __init__(self, config: dict, service_name: str,
                 sample_rate: float = 1.0,
                 trace_context: dict = None):
        """
        config: standard Kafka producer config
        service_name: name of this service (for headers)
        sample_rate: fraction of messages to trace (0.0 to 1.0)
        trace_context: optional context from consumed message
                       (for propagating trace_id through service chain)
        """
        self._producer = Producer(config)
        self.service_name = service_name
        self.sample_rate = sample_rate
        self.metrics = get_metrics(service_name)

        # Trace context from upstream (if this service also consumes)
        self._trace_context = trace_context or {}
        self._produce_times = {}

        print(f"  🔧 InstrumentedProducer initialized: {service_name}")

    def set_trace_context(self, context: dict):
        """
        Set trace context from a consumed message.
        Called by InstrumentedConsumer when a message is received
        so that the producer can propagate the trace_id.
        """
        self._trace_context = context

    def produce(self, topic: str, value=None, key=None,
                headers=None, callback=None, **kwargs):
        """
        Produce with automatic interceptor injection.
        Same signature as confluent_kafka.Producer.produce()
        """
        try:
            # ============================================
            # INTERCEPTOR: on_send
            # ============================================
            if headers is None:
                headers = []
            elif isinstance(headers, dict):
                headers = list(headers.items())

            # Trace ID: propagate from context or generate new
            existing_trace = HeaderUtils.get_header(headers, HeaderUtils.TRACE_ID)
            context_trace = self._trace_context.get('trace_id')

            if existing_trace:
                trace_id = existing_trace
            elif context_trace:
                trace_id = context_trace
            else:
                trace_id = HeaderUtils.generate_trace_id()

            headers = HeaderUtils.set_header(headers, HeaderUtils.TRACE_ID, trace_id)

            # Source service
            headers = HeaderUtils.set_header(
                headers, HeaderUtils.SOURCE_SERVICE, self.service_name
            )

            # Produce timestamp
            produce_ts = HeaderUtils.current_timestamp_ms()
            headers = HeaderUtils.set_header(
                headers, HeaderUtils.PRODUCE_TIMESTAMP, produce_ts
            )

            # Hop count: increment from context or start at 0
            context_hops = self._trace_context.get('hop_count', '0')
            try:
                new_hops = str(int(context_hops) + 1)
            except ValueError:
                new_hops = '1'
            headers = HeaderUtils.set_header(
                headers, HeaderUtils.HOP_COUNT, new_hops
            )

            # Correlation ID: propagate from context if exists
            context_corr = self._trace_context.get('correlation_id')
            if context_corr:
                headers = HeaderUtils.set_header(
                    headers, HeaderUtils.CORRELATION_ID, context_corr
                )

            # Sampling decision
            import random
            sampled = random.random() < self.sample_rate
            headers = HeaderUtils.set_header(
                headers, HeaderUtils.TRACE_SAMPLED,
                '1' if sampled else '0'
            )

            # Track produce time for acknowledgment latency
            internal_id = f"{trace_id}-{produce_ts}"
            self._produce_times[internal_id] = time.time()

            # Record metrics
            msg_size = len(value) if value else 0
            self.metrics.record_produce(topic, msg_size)

            # Record trace event
            if sampled:
                self.metrics.record_trace(
                    trace_id, self.service_name, 'produce',
                    topic, float(produce_ts)
                )

            # ============================================
            # ACTUAL PRODUCE (with wrapped callback)
            # ============================================
            original_callback = callback

            def wrapped_callback(err, msg):
                """Interceptor on_acknowledgement wrapper"""
                try:
                    if err:
                        self.metrics.record_error(topic, str(err))
                    else:
                        # Calculate acknowledgment latency
                        if internal_id in self._produce_times:
                            ack_latency = (time.time() - self._produce_times[internal_id]) * 1000
                            del self._produce_times[internal_id]
                except Exception:
                    pass  # Interceptor errors must never propagate

                # Call original callback
                if original_callback:
                    original_callback(err, msg)

            self._producer.produce(
                topic=topic,
                value=value,
                key=key,
                headers=headers,
                callback=wrapped_callback,
                **kwargs
            )

        except Exception as interceptor_error:
            # CRITICAL: interceptor failure must NOT block message delivery
            # Fall back to producing without interceptor
            print(f"  ⚠️  Interceptor error (non-blocking): {interceptor_error}")
            self._producer.produce(
                topic=topic, value=value, key=key,
                headers=headers, callback=callback, **kwargs
            )

    def poll(self, timeout=0):
        return self._producer.poll(timeout)

    def flush(self, timeout=None):
        if timeout is not None:
            return self._producer.flush(timeout)
        return self._producer.flush()

    def __len__(self):
        return len(self._producer)


class InstrumentedConsumer:
    """
    Drop-in replacement for confluent_kafka.Consumer.
    Transparently adds:
    → Latency calculation from produce_timestamp_ms
    → Trace ID extraction and logging
    → Message counting metrics
    → Trace context propagation for downstream producers
    
    Usage:
      # BEFORE:
      consumer = Consumer({'bootstrap.servers': 'localhost:9092', ...})
      
      # AFTER (ONLY this line changes):
      consumer = InstrumentedConsumer(
          {'bootstrap.servers': 'localhost:9092', ...},
          service_name='fraud-service'
      )
    """

    def __init__(self, config: dict, service_name: str,
                 linked_producer: Optional[InstrumentedProducer] = None):
        """
        config: standard Kafka consumer config
        service_name: name of this service
        linked_producer: optional InstrumentedProducer to propagate trace context
        """
        self._consumer = Consumer(config)
        self.service_name = service_name
        self.metrics = get_metrics(service_name)
        self.linked_producer = linked_producer
        self._last_trace_context = {}

        print(f"  🔧 InstrumentedConsumer initialized: {service_name}")

    def subscribe(self, topics, **kwargs):
        return self._consumer.subscribe(topics, **kwargs)

    def poll(self, timeout=1.0):
        """
        Poll with automatic interceptor processing.
        Same signature as confluent_kafka.Consumer.poll()
        """
        msg = self._consumer.poll(timeout=timeout)

        if msg is None:
            return None

        if msg.error():
            if msg.error().code() != KafkaError._PARTITION_EOF:
                self.metrics.record_error(
                    msg.topic() or 'unknown',
                    str(msg.error())
                )
            return msg

        try:
            # ============================================
            # INTERCEPTOR: on_consume
            # ============================================
            topic = msg.topic()
            headers = msg.headers() or []

            # Record consume metrics
            msg_size = len(msg.value()) if msg.value() else 0
            self.metrics.record_consume(topic, msg_size)

            # Extract trace context
            trace_id = HeaderUtils.get_header(headers, HeaderUtils.TRACE_ID)
            source_service = HeaderUtils.get_header(headers, HeaderUtils.SOURCE_SERVICE)
            produce_ts_str = HeaderUtils.get_header(headers, HeaderUtils.PRODUCE_TIMESTAMP)
            hop_count = HeaderUtils.get_header(headers, HeaderUtils.HOP_COUNT)
            correlation_id = HeaderUtils.get_header(headers, HeaderUtils.CORRELATION_ID)
            trace_sampled = HeaderUtils.get_header(headers, HeaderUtils.TRACE_SAMPLED)

            # Calculate produce-to-consume latency
            if produce_ts_str:
                try:
                    produce_ts = float(produce_ts_str)
                    consume_ts = time.time() * 1000
                    latency_ms = consume_ts - produce_ts

                    if latency_ms >= 0:
                        self.metrics.record_latency(topic, latency_ms)
                except (ValueError, TypeError):
                    pass

            # Record trace event
            if trace_sampled == '1' and trace_id:
                self.metrics.record_trace(
                    trace_id, self.service_name, 'consume',
                    topic, time.time() * 1000
                )

            # Store trace context for downstream propagation
            self._last_trace_context = {
                'trace_id': trace_id,
                'correlation_id': correlation_id,
                'hop_count': hop_count or '0',
                'source_service': source_service,
                'trace_sampled': trace_sampled,
            }

            # Propagate to linked producer
            if self.linked_producer:
                self.linked_producer.set_trace_context(self._last_trace_context)

        except Exception as interceptor_error:
            # CRITICAL: interceptor failure must NOT block message consumption
            print(f"  ⚠️  Consumer interceptor error (non-blocking): {interceptor_error}")

        return msg

    def get_trace_context(self) -> dict:
        """Get the trace context from the last consumed message"""
        return self._last_trace_context.copy()

    def commit(self, **kwargs):
        return self._consumer.commit(**kwargs)

    def close(self):
        return self._consumer.close()

    def assign(self, partitions):
        return self._consumer.assign(partitions)

from confluent_kafka import Consumer, KafkaError
from interceptor_framework import HeaderUtils
import json

class TraceInspector:
    """
    Reads messages from ALL topics and reconstructs
    the complete trace chain for each trace_id.
    
    This is the observability tool that answers:
    → Where is my payment RIGHT NOW?
    → How long did each hop take?
    → Which service is the bottleneck?
    → Did the message reach the final consumer?
    """

    def __init__(self, topics=None, broker='localhost:9092'):
        self.topics = topics or ['payments', 'fraud-results', 'orders', 'notifications']
        self.broker = broker

        self.consumer = Consumer({
            'bootstrap.servers': broker,
            'group.id': f'trace-inspector-{int(time.time())}',
            'auto.offset.reset': 'earliest',
            'enable.auto.commit': True,
        })
        self.consumer.subscribe(self.topics)

        # Trace storage: trace_id → list of events
        self.traces = defaultdict(list)
        self.running = True

    def collect_traces(self, duration_seconds=30):
        """Collect trace events from all topics"""
        print(f"\n  🔍 TRACE INSPECTOR — Collecting from {', '.join(self.topics)}")
        print(f"     Duration: {duration_seconds}s")

        start = time.time()

        while self.running and time.time() - start < duration_seconds:
            msg = self.consumer.poll(timeout=1.0)
            if msg is None:
                continue
            if msg.error():
                if msg.error().code() == KafkaError._PARTITION_EOF:
                    continue
                continue

            try:
                topic = msg.topic()
                headers = msg.headers() or []
                value = json.loads(msg.value().decode('utf-8')) if msg.value() else {}

                trace_id = HeaderUtils.get_header(headers, HeaderUtils.TRACE_ID)
                if not trace_id:
                    continue

                source = HeaderUtils.get_header(headers, HeaderUtils.SOURCE_SERVICE)
                produce_ts = HeaderUtils.get_header(headers, HeaderUtils.PRODUCE_TIMESTAMP)
                hop_count = HeaderUtils.get_header(headers, HeaderUtils.HOP_COUNT)
                correlation_id = HeaderUtils.get_header(headers, HeaderUtils.CORRELATION_ID)

                event = {
                    'topic': topic,
                    'source_service': source or '?',
                    'produce_timestamp_ms': float(produce_ts) if produce_ts else 0,
                    'hop_count': int(hop_count) if hop_count else 0,
                    'correlation_id': correlation_id,
                    'kafka_timestamp': msg.timestamp()[1] if msg.timestamp() else 0,
                    'partition': msg.partition(),
                    'offset': msg.offset(),
                    'value_summary': self._summarize_value(value),
                    'all_headers': HeaderUtils.headers_to_dict(headers),
                }

                self.traces[trace_id].append(event)

            except Exception as e:
                pass  # Skip unparseable messages

        self.consumer.close()
        print(f"  ✅ Collected {sum(len(v) for v in self.traces.values())} events "
              f"across {len(self.traces)} traces")

    def _summarize_value(self, value: dict) -> str:
        """Create short summary of message value"""
        if 'payment_id' in value and 'amount' in value:
            return f"Payment ${value['amount']} ({value.get('status', '?')})"
        elif 'fraud_check' in value:
            return f"Fraud: {value['fraud_check']} (risk: {value.get('risk_score', '?')})"
        elif 'order_id' in value:
            return f"Order {value['order_id']} ({value.get('status', '?')})"
        else:
            keys = list(value.keys())[:3]
            return f"Keys: {', '.join(keys)}"

    def print_trace(self, trace_id: str):
        """Print detailed visualization of a single trace"""
        events = self.traces.get(trace_id, [])
        if not events:
            print(f"  ❌ Trace {trace_id} not found")
            return

        # Sort by timestamp
        events.sort(key=lambda e: e['produce_timestamp_ms'])

        first_ts = events[0]['produce_timestamp_ms']
        last_ts = events[-1]['produce_timestamp_ms']
        total_duration = last_ts - first_ts if last_ts > first_ts else 0

        correlation = events[0].get('correlation_id', '?')

        print(f"\n  ┌{'─'*66}┐")
        print(f"  │ TRACE: {trace_id:<56s}  │")
        print(f"  │ Correlation: {str(correlation):<50s}  │")
        print(f"  │ Events: {len(events):<4d}  Duration: {total_duration:.0f}ms{' '*28}│")
        print(f"  ├{'─'*66}┤")

        for i, event in enumerate(events):
            relative_ms = event['produce_timestamp_ms'] - first_ts
            hop = event['hop_count']
            indent = '  ' * hop

            # Calculate inter-hop latency
            if i > 0:
                prev_ts = events[i-1]['produce_timestamp_ms']
                hop_latency = event['produce_timestamp_ms'] - prev_ts
                latency_str = f" (+{hop_latency:.0f}ms)"
            else:
                latency_str = " (entry)"

            service = event['source_service']
            topic = event['topic']
            summary = event['value_summary']

            print(f"  │ [{relative_ms:>6.0f}ms] {indent}📤 {service:<20s} → {topic}{latency_str}")
            print(f"  │          {indent}   {summary}")

        else:

        print(f"  │ Total Duration: {total_duration:.0f}ms{' '*(48 - len(str(int(total_duration))))}│")
        print(f"  └{'─'*66}┘")

    def print_all_traces(self, max_traces=20):

        print(f"\n  {'='*70}")
        print(f"  🔗 ALL COLLECTED TRACES ({len(self.traces)} total)")

        if not self.traces:

            return

        print(f"\n  {'Trace ID':<40s} {'Events':<8s} {'Hops':<6s} "
              f"{'Duration':<12s} {'Status'}")
        print(f"  {'─'*40} {'─'*8} {'─'*6} {'─'*12} {'─'*15}")

        sorted_traces = sorted(
            self.traces.items(),
            key=lambda x: x[1][0]['produce_timestamp_ms'] if x[1] else 0,

        )

        for trace_id, events in sorted_traces[:max_traces]:
            events.sort(key=lambda e: e['produce_timestamp_ms'])

            num_events = len(events)
            max_hop = max(e['hop_count'] for e in events) if events else 0

            first_ts = events[0]['produce_timestamp_ms']
            last_ts = events[-1]['produce_timestamp_ms']
            duration = last_ts - first_ts

            topics_seen = set(e['topic'] for e in events)
            if 'orders' in topics_seen:
                status = "✅ Completed"
            elif any('FAIL' in e.get('value_summary', '') for e in events):
                status = "🚫 Rejected"
            else:
                status = "⏳ In Progress"

            short_id = trace_id[:36] + '...' if len(trace_id) > 36 else trace_id
            print(f"  {short_id:<40s} {num_events:<8d} {max_hop:<6d} "
                  f"{duration:>8.0f}ms   {status}")

        # Aggregate stats
        all_durations = []
        completed = 0
        rejected = 0
        in_progress = 0

        for trace_id, events in self.traces.items():
            if not events:
                continue
            events.sort(key=lambda e: e['produce_timestamp_ms'])
            duration = events[-1]['produce_timestamp_ms'] - events[0]['produce_timestamp_ms']
            all_durations.append(duration)

            topics_seen = set(e['topic'] for e in events)
            if 'orders' in topics_seen:
                completed += 1
            elif any('FAIL' in e.get('value_summary', '') for e in events):
                rejected += 1
            else:
                in_progress += 1

        print(f"\n  📊 AGGREGATE STATISTICS:")
        print(f"     Total traces:  {len(self.traces)}")
        print(f"     Completed:     {completed}")
        print(f"     Rejected:      {rejected}")
        print(f"     In progress:   {in_progress}")

        if all_durations:

            count = len(all_durations)

            print(f"     Duration p95:  {all_durations[int(count * 0.95)]:.0f}ms")
            print(f"     Duration p99:  {all_durations[min(int(count * 0.99), count - 1)]:.0f}ms")
            print(f"     Duration avg:  {sum(all_durations) / count:.0f}ms")

    def search_by_correlation(self, correlation_id: str):
        """Find traces by business correlation ID (e.g., payment_id)"""
        matching = []
        for trace_id, events in self.traces.items():
            for event in events:
                if event.get('correlation_id') == correlation_id:
                    matching.append(trace_id)
                    break

        if not matching:
            print(f"  ❌ No traces found for correlation_id: {correlation_id}")
            return

        print(f"  🔍 Found {len(matching)} trace(s) for correlation: {correlation_id}")
        for tid in matching:
            self.print_trace(tid)

    def find_slow_traces(self, threshold_ms: float = 500):
        """Find traces exceeding latency threshold"""
        slow = []
        for trace_id, events in self.traces.items():
            if not events:
                continue
            events.sort(key=lambda e: e['produce_timestamp_ms'])
            duration = events[-1]['produce_timestamp_ms'] - events[0]['produce_timestamp_ms']
            if duration > threshold_ms:
                slow.append((trace_id, duration))

        print(f"\n  🐌 SLOW TRACES (>{threshold_ms}ms): {len(slow)} found")
        for tid, duration in slow[:10]:
            print(f"     {tid[:36]}... → {duration:.0f}ms")

    def find_incomplete_traces(self):

        for tid in incomplete[:10]:
            events = self.traces[tid]

            last_service = events[-1]['source_service'] if events else '?'
            print(f"     {tid[:36]}... → stuck at {last_service} ({last_topic})")

from http.server import HTTPServer, BaseHTTPRequestHandler
import threading
import time

class PrometheusExporter:
    """
    Exposes interceptor metrics in Prometheus exposition format.
    Run alongside your services — Prometheus scrapes this endpoint.
    
    Connects to Project 28 (Monitoring):
    → Prometheus scrapes /metrics endpoint
    → Grafana dashboards visualize the data
    → Alerts on latency threshold breaches
    """

    def __init__(self, port=9090):
        self.port = port

    def _generate_metrics(self) -> str:
        """Generate Prometheus exposition format metrics"""
        lines = []
        all_metrics = get_all_metrics()

        for service_name, snapshot in all_metrics.items():
            safe_service = service_name.replace('-', '_')

            # Produce counts
            lines.append(f"# HELP kafka_interceptor_produced_total Messages produced by service")
            lines.append(f"# TYPE kafka_interceptor_produced_total counter")
            for topic, count in snapshot.get('produce', {}).items():
                safe_topic = topic.replace('-', '_')
                lines.append(
                    f'kafka_interceptor_produced_total{{service="{service_name}",'
                    f'topic="{topic}"}} {count}'
                )

            # Consume counts
            lines.append(f"# HELP kafka_interceptor_consumed_total Messages consumed by service")
            lines.append(f"# TYPE kafka_interceptor_consumed_total counter")
            for topic, count in snapshot.get('consume', {}).items():
                lines.append(
                    f'kafka_interceptor_consumed_total{{service="{service_name}",'
                    f'topic="{topic}"}} {count}'
                )

            # Produce rates
            lines.append(f"# HELP kafka_interceptor_produce_rate Messages per second")
            lines.append(f"# TYPE kafka_interceptor_produce_rate gauge")
            for topic, rate in snapshot.get('produce_rates', {}).items():
                lines.append(
                    f'kafka_interceptor_produce_rate{{service="{service_name}",'
                    f'topic="{topic}"}} {rate}'
                )

            # Latency percentiles
            lines.append(f"# HELP kafka_interceptor_latency_ms Produce-to-consume latency")
            lines.append(f"# TYPE kafka_interceptor_latency_ms gauge")
            for topic, stats in snapshot.get('latency', {}).items():
                for pct in ['p50', 'p95', 'p99', 'avg']:
                    lines.append(
                        f'kafka_interceptor_latency_ms{{service="{service_name}",'
                        f'topic="{topic}",quantile="{pct}"}} {stats[pct]:.2f}'
                    )

            # Errors
            lines.append(f"# HELP kafka_interceptor_errors_total Error count")
            lines.append(f"# TYPE kafka_interceptor_errors_total counter")
            for key, count in snapshot.get('errors', {}).items():
                lines.append(
                    f'kafka_interceptor_errors_total{{service="{service_name}",'
                    f'error="{key}"}} {count}'
                )

        return '\n'.join(lines) + '\n'

    def start(self):
        """Start the Prometheus metrics HTTP server"""
        exporter = self

        class MetricsHandler(BaseHTTPRequestHandler):
            def do_GET(self):
                if self.path == '/metrics':
                    metrics = exporter._generate_metrics()
                    self.send_response(200)
                    self.send_header('Content-Type', 'text/plain; charset=utf-8')
                    self.end_headers()
                    self.wfile.write(metrics.encode('utf-8'))
                elif self.path == '/health':
                    self.send_response(200)
                    self.send_header('Content-Type', 'application/json')
                    self.end_headers()
                    self.wfile.write(b'{"status": "healthy"}')
                else:
                    self.send_response(404)
                    self.end_headers()

            def log_message(self, format, *args):
                pass  # Suppress HTTP access logs

        server = HTTPServer(('0.0.0.0', self.port), MetricsHandler)
        thread = threading.Thread(target=server.serve_forever, daemon=True)
        thread.start()
        print(f"  📊 Prometheus exporter running on http://localhost:{self.port}/metrics")
        return server


if __name__ == '__main__':
